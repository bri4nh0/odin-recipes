# odin-recipes
This project is to demonstrate basic HTML skills. The project will be in the form of a basic recipe website which utilizes a main index page and several relative pages.

The index page contains three links to three different recipes:
- No-Mayo Egg Salad
- Strawberry Rhubarb Drop Biscuits
- Peanut Butter Oatmeal

Each recipe page contains a short description, ingredients list, and steps to make the meal.
